**E-Shop**

# Overview

Simple E-commerce APP UI. Users can check shipping details, packing details, date of order, the status of the product etc...



# Admin features
1. Dashboard
2. Shipping
3. Inventory
4. Orders
5. Channel


# Status features
1. Pending
2. Accepted
3. AWB Created
4. Ready to ship
5. Shipped
6. Completed
7. Cancelled

# Technologies used
1. React Js
2. Javascript
3. Jsx
4. CSS
5. React Icons
6. React + Vite Framework

# Installation guide
1. git clone repository link
2. cd folder name
3. npm install
4. npm run dev
